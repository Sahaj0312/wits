FLOW SUMMARY
=====================================================
Account creation → Goals (multi-select) → 6 self-assessment Likert questions →
Demographics (gender, education, attribution) → Fit Test intro →
3 calibration games (Color Match / Flexibility, Train of Thought / Attention,
Memory Matrix / Memory), each with intro + tutorial + live play + explainer →
Fit Test Results (percentile vs. others) → Daily Streak + reminders (engagement
hooks) → Program builder questionnaire (difficulty, motivation, exercise, sleep,
training days) → plan-generation loading → 30-Day Plan projection (LPI +44%) →
Paywall (7-day trial, 3 pricing tiers).

KEY PATTERNS TO NOTE:
- Progress bar shown across question screens to signal forward momentum.
- Heavy personalization framing ("build the right plan for you").
- Self-assessment primes the user to feel a cognitive problem before testing.
- Games double as engagement (fun) + calibration (baseline scores).
- Results use social comparison ("higher than X% of people").
- Engagement hooks (streak, reminders) placed BEFORE the paywall.
- Projected improvement (+44% LPI) builds value/urgency right before pricing.
- Paywall leads with a free trial and a pre-selected "most popular" annual plan.
