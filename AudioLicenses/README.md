# Third-party audio

The application includes a curated and renamed subset of the following CC0
sound library. The source filenames are retained below so every shipped file
can be traced back to its author and license.

## Dustyroom — Casual Game Sounds

- Source: https://dustyroom.com/free-casual-game-sounds/
- License: CC0 1.0 Universal
- Original files used:
  - `DM-CGS-06.wav` → `game_over.wav`
  - `DM-CGS-18.wav` → `new_best.wav`
  - `DM-CGS-28.wav` → `life_lost.wav`
  - `DM-CGS-33.wav` → `level_up.wav`
  - `DM-CGS-34.wav` → `correct_1.wav`
  - `DM-CGS-38.wav` → `combo.wav`
  - `DM-CGS-44.wav` → `correct_2.wav`

All samples were converted to mono 44.1 kHz 16-bit PCM WAV and peak-normalized
to -6 dBFS for consistent gain staging in the application.
CC0 permits copying, modification, distribution, and commercial use without
attribution. Canonical license: https://creativecommons.org/publicdomain/zero/1.0/
